# แบบฝึก AR อย่างง่ายบนมือถือ

## ชื่อกิจกรรม
**AR Motor Status — ตรวจสอบสถานะมอเตอร์เสมือน**

## เทคโนโลยีที่ใช้
- A-Frame 1.6.0
- AR.js 3.4.7
- HTML, CSS และ JavaScript
- ใช้งานผ่านเว็บเบราว์เซอร์บนมือถือ ไม่ต้องสร้างแอป Android หรือ iOS

## ระยะเวลา
ประมาณ 45–60 นาที

## ผลลัพธ์การเรียนรู้
เมื่อจบกิจกรรม ผู้เรียนสามารถ:
1. อธิบายหลักการ Marker-based AR ได้
2. เปิด AR ผ่านกล้องมือถือได้
3. สร้างวัตถุ 3 มิติจาก primitive ของ A-Frame
4. เปลี่ยนสี ข้อความ และความเร็วการหมุนด้วย HTML/JavaScript
5. เผยแพร่เว็บ AR บนบริการ Static Web Hosting ที่มี HTTPS ได้

## ไฟล์ในชุดนี้
- `index.html` โปรแกรม AR
- `hiro-marker.png` รูป Marker สำหรับพิมพ์หรือแสดงบนอีกหน้าจอ
- `WORKSHEET_TH.md` ใบงานสำหรับผู้เรียน
- `LICENSE.txt` เงื่อนไขการนำตัวอย่างไปใช้

# วิธีทดลองที่ง่ายที่สุดบนมือถือ

## วิธีที่ 1: ใช้ GitHub Pages

1. สร้างบัญชี GitHub และสร้าง Repository ใหม่ เช่น `mobile-ar-lab`
2. Upload ไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น Repository
3. เปิด `Settings` → `Pages`
4. ในหัวข้อ Source เลือก `Deploy from a branch`
5. เลือก Branch เป็น `main` และ Folder เป็น `/root`
6. กด Save และรอจนได้ URL ของเว็บไซต์
7. เปิด URL นั้นด้วย Chrome บน Android หรือ Safari บน iPhone
8. กดอนุญาตให้เว็บไซต์ใช้กล้อง
9. ใช้กล้องส่องไปที่ `hiro-marker.png`

> กล้องของเว็บเบราว์เซอร์ควรเปิดจากเว็บไซต์ที่ใช้ HTTPS หรือจาก localhost

## วิธีที่ 2: ทดลองบนคอมพิวเตอร์ก่อน

เปิด Terminal หรือ Command Prompt ที่โฟลเดอร์นี้ แล้วรัน:

```bash
python -m http.server 8000
```

จากนั้นเปิด:

```text
http://localhost:8000
```

วิธีนี้เหมาะสำหรับตรวจสอบโค้ดบนคอมพิวเตอร์ แต่โทรศัพท์เครื่องอื่นจะต้องเข้าผ่านเว็บที่มี HTTPS เพื่อให้การขอใช้กล้องทำงานได้สม่ำเสมอ

# วิธีใช้งาน

1. วางหรือพิมพ์ Hiro Marker ให้ราบ
2. เปิดหน้าเว็บ AR บนมือถือ
3. ให้ Marker อยู่ในภาพครบทั้งกรอบดำ
4. เมื่อระบบตรวจพบ Marker จะเห็นโมเดลมอเตอร์เหนือ Marker
5. กด `หยุดมอเตอร์/เริ่มมอเตอร์` เพื่อควบคุมเพลา
6. กด `จำลองคำเตือน/เปลี่ยนเป็นปกติ` เพื่อเปลี่ยนสีและข้อความสถานะ

# จุดที่ให้ผู้เรียนแก้ไข

## 1. เปลี่ยนสีมอเตอร์
ค้นหาบรรทัด:

```html
color="#1976d2"
```

เปลี่ยนเป็นสีอื่น เช่น:

```html
color="#ff9800"
```

## 2. เปลี่ยนชื่อเครื่องจักร
ค้นหา:

```html
value="MOTOR M1\nSTATUS: NORMAL"
```

เปลี่ยน `MOTOR M1` เป็นชื่อที่ต้องการ เช่น `PUMP P-101`

## 3. เปลี่ยนความเร็วเพลา
ค้นหา:

```html
spin-control="axis: y; speed: 220; active: true"
```

ลองเปลี่ยน `speed: 220` เป็น `speed: 80` หรือ `speed: 500`

## 4. เพิ่มชิ้นส่วน
เพิ่มกรวยบนฐาน:

```html
<a-cone
  position="0.45 0.35 0.28"
  radius-bottom="0.12"
  radius-top="0.04"
  height="0.35"
  color="#ff9800"
></a-cone>
```

ให้นำโค้ดนี้ไปวางภายใน `<a-marker> ... </a-marker>`

# การแก้ปัญหาเบื้องต้น

## กล้องไม่เปิด
- ตรวจสอบว่า URL เริ่มด้วย `https://`
- อนุญาต Camera Permission ให้เบราว์เซอร์
- ปิดแท็บแล้วเปิดใหม่หลังเปลี่ยน Permission

## เห็นกล้องแต่ไม่เห็นโมเดล
- ให้เห็นกรอบดำของ Marker ครบ
- เพิ่มแสงในห้อง แต่หลีกเลี่ยงแสงสะท้อนบนกระดาษ
- ถือมือถือห่าง Marker ประมาณ 20–50 ซม.
- อย่าพับ ตัด หรือบังส่วนใดของ Marker

## โมเดลสั่น
- วาง Marker ให้ราบและนิ่ง
- ใช้ Marker ขนาดประมาณ 10–15 ซม.
- เพิ่มแสงให้สม่ำเสมอ

## ปุ่มอยู่บนจอแต่กดไม่ได้
- รอให้หน้าเว็บโหลดครบ
- ตรวจสอบว่าไม่ได้เปิดหน้าเว็บในโหมด Preview ของบางแอป ให้เปิดใน Chrome หรือ Safari โดยตรง
